[RU]

Установка

1. Скопировать файлы из папки upload в корень сайта.
2. Перейти в раздел "Система" "Пользователи" "Группы пользователей" "Главный администратор" и 
разрешить просмотр и редактирование администратором раздела "extension/extended_module". Затем сохраните настройки.
3. Готово, перейдите в раздел модулей.

[EN]

Installation

1. Copy the files from the "upload" direcory to the root directory of the site.
2. Go to "System", "Users", "User Groups" "Chief Administrator" and
allow you to view and edit the administrator section "extension / extended_module". Then save the settings.
3. Finish, go to modules.
